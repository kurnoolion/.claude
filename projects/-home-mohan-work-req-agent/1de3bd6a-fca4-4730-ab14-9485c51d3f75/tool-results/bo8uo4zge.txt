Data retry requirements for LTE are specified across various scenarios and procedures, including:

**General Principles and Error Handling:**

*   The retry logic is designed to prevent User Equipment (UEs) from overwhelming the network with ineffective access attempts (VZ_REQ_LTEDATARETRY_68).
*   Devices may be required to throttle or, in extreme cases, disable access attempts if specific permanent or semi-permanent errors exist (VZ_REQ_LTEDATARETRY_68).
*   Specific requirements cover data retry for situations where the UE encounters RRC errors (VZ_REQ_LTEDATARETRY_2383), and for conditions encountered during EMM connection management procedures (VZ_REQ_LTEDATARETRY_23888).
*   Additional retry requirements are defined for conditions encountered during EMM specific procedures (VZ_REQ_LTEDATARETRY_7746) and EMM common procedures (VZ_REQ_LTEDATARETRY_7790).

**Specific Failure Scenarios:**

*   **Attach Attempt Counter Reaches 5:** If the attach attempt counter reaches a value of 5, the UE must:
    *   Follow the procedure detailed in 3GPP TS 24.301, section 5.5.1.2.6 (VZ_REQ_LTEDATARETRY_7756).
    *   Start timer T3402 and must not attempt to attach to LTE on that PLMN again until T3402 has expired (VZ_REQ_LTEDATARETRY_7756).
    *   Indicate a failure to the application requesting the attachment (VZ_REQ_LTEDATARETRY_7756).
    *   If T3402 is running, the UE should attach to another LTE network associated with a different PLMN if system selection indicates so (VZ_REQ_LTEDATARETRY_7756).
*   **Authentication Reject Message:** If the network sends an 'Authentication Reject' message to the UE, the UE must:
    *   Follow the steps detailed in 3GPP TS 24.301, section 5.4.2 (VZ_REQ_LTEDATARETRY_7744).
    *   Cease attempting to attach to any LTE network until it is power-cycled or the UICC is replaced (VZ_REQ_LTEDATARETRY_7744).
*   **Service Request Attempt Counter:** For LTE Release 12 and later UEs, the UE must support a service request attempt counter as detailed in 3GPP TS 24.301, Release 12, section 5.6.1.1 (VZ_REQ_LTEDATARETRY_23889).
